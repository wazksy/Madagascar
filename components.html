    Madagascar

    Trozos de código HTML reutilizables

    Madagscar cuenta con un conjunto de componentes compuestos para ser reutilizados o incluidos como shortcodes de WordPress. Utilizan reglas CSS específicas o clases genéricas.

    Icons

    Madagascar incluye la librería de iconos Awesome.

    Basta con añadir la clase .icon más la clase del icono en cuestión .icon--fa-laptop a cualquier etiqueta, aunque el resultado puede ser inesperado. Normalmente se utiliza la etiqueta i con poco valor semántico.

    Puede modificar el tamaño del icono añadiendo las clases .icon--xs, .icon--s, .icon--m, .icon--l o .icon--xl.

    También puedes modificar el aspecto de un icono añadiendo un círculo de fondo con la clase .icon--circle o un border circular con la clase .icon--border
